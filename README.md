
# Практикум по проектному менеджменту

Данный репозиторий содержит результаты выполнения практических работ по проектированию, оценке и планированию программного проекта.

## Команда проекта
*   **Пекшев**
*   **Ряховский**
*   **Эксперт А** (наш третий коллега)

---

## Выбранный проект
**Наименование:** Онлайн-магазин тортов с интерактивным 3D-конструктором.
**Суть:** Веб-платформа, позволяющая пользователям проектировать дизайн торта (ярусы, форма, декор) в 3D с автоматическим расчетом стоимости и интеграцией с производственным циклом.

---

## Навигация по фазам работ

1.  **[Фаза 1: Выбор концепции проекта](Фаза_1/Фаза_1.md)**
    *   Сравнение 4-х альтернативных идей методами МАИ и Fuzzy TOPSIS. Обоснование выбора текущего проекта.
2.  **[Фаза 2: Распределение работы (Мета-задача)](Фаза_2/Фаза_2.md)**
    *   Математическая оптимизация распределения задач между участниками на основе предпочтений и сложности (линеаризация задачи в Solver).
3.  **[Фаза 3: Оценка проекта](Фаза_3/Фаза_3.md)**
    *   Детальный расчет масштаба системы методами COSMIC Function Points, Use Case Points и коллективный Planning Poker.
4.  **[Фаза 4: Прогнозирование](Фаза_4/Фаза_4.md)**
    *   Имитационное моделирование сроков разработки методом Монте-Карло на основе исторической производительности команды.
5.  **[Фаза 5: Планирование](Фаза_5/Фаза_5.md)**
    *   Формирование детального календарного графика работ с использованием эвристических методов упорядочивания при ограниченных ресурсах.
